# Architecture-des-ordinateurs
C'est un projet pédagogique fournissant un noyau RISC minimal et son émulateur en C. Il comprend un simulateur (emulator) gérant la mémoire, l'exécution d'instructions et les périphériques simulés, ainsi que des exemples de logiciels embarqués (embedded_software) avec Makefiles, script d'édition de liens et routines système.
